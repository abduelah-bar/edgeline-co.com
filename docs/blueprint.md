# **App Name**: EnVision Engineering

## Core Features:

- Homepage Showcase: Display key project metrics like '138+' and '4,283' with sleek animations on the landing page.
- Service Overview: Present the company's core services such as 'Process Engineering' and 'Civil Engineering' with brief descriptions and visuals.
- Project Gallery: Exhibit a portfolio of completed projects using a grid layout, highlighting key details for each.
- Global Project Map: Visually represent the company's global presence using a world map with project locations and key statistics.
- Team Introduction: Introduce the leadership team with professional photos and brief biographies, showcasing expertise and experience.
- FAQ Section: Provide an interactive FAQ section to address common customer queries and improve user experience.
- Blog Integration: Pull the latest blog articles to dynamically display previews in the bottom section of the main page.

## Style Guidelines:

- Primary color: Deep blue (#003049) to convey trust and stability.
- Background color: Very dark desaturated blue (#033649) to provide a professional and calming backdrop.
- Accent color: Yellow-gold (#D6C239) for CTAs and important information to add a touch of sophistication.
- Body and headline font: 'Inter', a sans-serif, for a clean, modern, and highly readable appearance throughout the site.
- Use a grid-based layout to organize content effectively and ensure a consistent look across all sections.
- Incorporate minimalist, line-based icons in yellow-gold to represent services and features.
- Employ subtle scrolling animations to guide users through the page and create a dynamic feel.
